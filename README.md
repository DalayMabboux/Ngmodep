## Synopsis

Generate a simple dependency graph of Angular modules (JavaScript (ES6)).
![Dependencygraph](Example.png "Example dependecy graph")

## Code Example

Ngmodep . dependecy.dot

## Motivation

Haskell learning toy project.

## Installation

stack install

## Tests

stack test
